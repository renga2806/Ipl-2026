Rules & points

  • All the paricipants are given a unique PIN, you have to use it to vote
  • If you forgot or want the PIN to be changed to something easy for you to remember, reach out to the admin

  • If a match is abandoned / no result, everyone who voted gets 1 point.

  • Once a match is locked, votes cannot be changed.
  • Leaderboard is updated after the admin marks the result.
  • If multiple users have the same score, they share the same highlight tier.
